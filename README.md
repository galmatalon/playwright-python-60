# 🎭 Playwright Python Automation Framework

A modern UI Automation Testing Framework built with **Python**, **Playwright**, **Pytest**, and the **Page Object Model (POM)** design pattern.

This project demonstrates best practices for building scalable, maintainable, and reusable UI automation frameworks suitable for real-world applications.

---

# 🚀 Technologies

* Python 3.x
* Playwright
* Pytest
* Page Object Model (POM)
* Pytest Fixtures
* HTML Reports
* Logging
* Git & GitHub

---

# 📁 Project Structure

```
playwright-python-60/
│
├── pages/                  # Page Object classes
├── tests/                  # Test cases
├── utilities/              # Helper functions
├── config/                 # Configuration files
├── reports/                # HTML reports
├── screenshots/            # Failure screenshots
├── conftest.py             # Global Pytest fixtures
├── pytest.ini              # Pytest configuration
├── requirements.txt
└── README.md
```

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/galmatalon/playwright-python-60.git
```

Move into the project

```bash
cd playwright-python-60
```

Install dependencies

```bash
pip install -r requirements.txt
```

Install Playwright browsers

```bash
playwright install
```

---

# ▶️ Running Tests

Run all tests

```bash
pytest
```

Run a specific test

```bash
pytest tests/test_example.py
```

Run with verbose output

```bash
pytest -v
```

Generate HTML Report

```bash
pytest --html=reports/report.html
```
